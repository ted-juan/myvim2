vim for C/C++ developer
====

my vimrc and plugin


Installation:

	git clone https://github.com/ted-juan/myvim2.git ~/.vim

Create symlinks:

	ln -s ~/.vim/.vimrc ~/.vimrc

Get Vundle plugin

	git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle

Use Bundle Install plugin
	    
	vim ~/.vimrc
	:BundleInstall!
